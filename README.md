### Prerequisites

Tool'un düzgün çalışması için ;

* Terminal'e;

```sh
apt-get update
```

### Installation

 Kali Linux Terminalini Açınız.

```sh
pip3 install gerekli.txt
```

```sh
git clone https://github.com/mxrkur1337/mxrkur-sms
```

```sh
cd mxrkur-sms
```

```sh
python3 mxrkursms.py
```
